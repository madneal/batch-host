## 安装chrome extension

通过文件夹加载extension，经常会出现提示，关闭开发者模式，所以将chrome extension打包进行安装。安装步骤：

* 打开chrome://extensions
* 然后直接将danmu-sender.crx文件拖到页面就可以了

*PS*

现在谷歌已经不允许非应用商店的程序通过上述安装方式安装了，所以只能通过开发者模式来安装：

* 打开chrome://extensions
* 勾选上开发者模式
* 加载解压的拓展程序，选择拓展程序所在文件夹
